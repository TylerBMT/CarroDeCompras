# 🎟️ Carro de Compras de Entradas — Página HTML

Landing / página web hecha en **HTML + CSS** que simula la compra de entradas para un concierto, incluyendo un **carro de compras** para seleccionar tickets, ajustar cantidades y ver el total.

---

## ✨ Funcionalidades

- 🛒 Carro de compras de entradas
- ➕➖ Ajuste de cantidad de tickets
- 💲 Cálculo de total
- 🧾 Resumen de compra en la misma página
- 🎨 Estilos personalizados con CSS

>`localStorage` para el guardado de datos pre-compra

---

## 🧰 Tecnologías

- **HTML5**
- **CSS3**
- **JavaScript**
- Imágenes en formato `.webp/.png/.jpg`
